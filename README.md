# Student Management App

A simple Student Management App built with Vanilla JavaScript.

## Features

- Add new students
- Search students by name
- Delete students
- Form validation
- Dynamic table rendering
- Array and object-based data management

## Technologies Used

- HTML
- CSS
- JavaScript

## Concepts Practiced

- DOM Manipulation
- Event Handling
- Arrays and Objects
- Array.filter()
- Dynamic Rendering
- Form Validation
